# Cocin Rashik Backend

This is the backend server for the Cocin Rashik application.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/)

## Installation

1. Clone the repository.
2. Install the dependencies:

   ```bash
   npm install
   ```

## Running the Application

To start the server:

```bash
npm start
```

To run in development mode (if nodemon is configured):

```bash
npm run dev
```

## Validation

This project uses Joi for input validation to ensure data integrity for API endpoints.
